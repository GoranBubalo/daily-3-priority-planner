# Daily 3 Focus Tasks

A productivity planner app that helps you **prioritize and complete your three most meaningful tasks each day**.  
This project is inspired by science‑backed focus methods that suggest limiting your daily priorities to three high‑impact items to increase focus, reduce overwhelm, and build better habits. :contentReference[oaicite:1]{index=1}

This app provides a simple, intuitive interface where users can:
- Select **three meaningful tasks** per day  
- Track completion status  
- View history of past focus days  
- See simple stats and streaks (planned)

The goal of this project is to learn full‑stack development while building something useful and *showcasing real skills* in frontend, backend, database, and deployment.

---

## 🚀 Features (initial)
- User authentication (sign up / login)
- Create, edit, and complete daily focus tasks
- Only three focus slots per day
- Nice responsive UI

## 🧠 Why this project?
This app goes beyond a typical to‑do list — it enforces a focus mindset by design. Giving users exactly three high‑priority slots encourages discipline and deeper work.
